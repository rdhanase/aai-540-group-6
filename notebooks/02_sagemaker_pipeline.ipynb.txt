# NYC Collision Severity Prediction - SageMaker Pipeline

This notebook provides a template for orchestrating the machine learning pipeline on AWS SageMaker.

## 1. Setup
- Import SageMaker SDK.
- Define S3 bucket and prefix.
- Define IAM execution role.

## 2. Data Ingestion & S3 Upload
- Use `data_loader.py` to fetch data.
- Upload raw data to S3: `s3://{bucket}/{prefix}/data/raw/`.

## 3. SageMaker Processing Step
- Create a `SKLearnProcessor`.
- Run a processing job using `preprocessing.py` and `features.py`.
- Output: Processed training/test data to S3.

## 4. SageMaker Training Step
- Use `SKLearn` Estimator with `sagemaker_train.py`.
- Input: Processed data from S3.
- Hyperparameters: `n_estimators`, `max_depth`.

## 5. Model Registration
- Register the trained model in the SageMaker Model Registry.
- Include metrics (Recall, Precision).

## 6. SageMaker Pipeline (DAG)
- Define the `Pipeline` object connecting the Processing, Training, and Registration steps.
- Execute the pipeline.
